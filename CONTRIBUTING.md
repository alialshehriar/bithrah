# Contributing to Bithrah

Thank you for your interest in contributing to Bithrah! 🌱

## About Bithrah

Bithrah (بذرة) is a Saudi AI-powered platform for idea empowerment through rewards and communities. We're currently in pre-launch phase, accepted to impactU by STC.

## How to Contribute

### Report Issues
- Open an issue describing the problem
- Include steps to reproduce
- Add screenshots if applicable

### Suggest Features
- Open an issue with the "feature request" label
- Describe the feature and its benefit
- Explain how it aligns with Bithrah's mission

### Code Contributions
1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Code of Conduct

- Be respectful and inclusive
- Support Arabic and English equally
- Follow existing code patterns
- Write clear commit messages

## Tech Stack

- Frontend: Next.js 15, React 19, TypeScript, TailwindCSS
- Backend: FastAPI, PostgreSQL
- AI/ML: Custom models
